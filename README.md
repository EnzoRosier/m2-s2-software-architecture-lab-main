# End of Module Project

## You need : 
node.js, nestjs

## Installation : 
Clone from github, run "npm i" from the root of the project, create a .env file with the following structure : 
PORT=
DATABASE_URL=
NODE_ENV=

You can also run "npm run seed" create a database with basic informations

run "npm run build"
